## Student Notes
If you have any bonus specs, bonus or any details the TA's should know, you should include it here:

-- MULTIPLAYER 1V1 MODE --
Once you run the program, to activate the multiplayer 1v1 bonus mode, open the 'Startup' menu and then press the 'M' key. 
You will then see a single enemy that the second player can control using the arrow keys:
Press the "Left Arrow Key" to move left, press the "Right Arrow Key" to move right, and press the "Up Arrow Key" to attack.

Additionally, besides editing some already existing files, two new files were created to implement the bonus: PlayerEnemy.hpp and PlayerEnemy.cpp. These files are located in the following directory: src folder → Entities folder → Enemies folder.
(To make it easier to identify which parts of the programming relate to the bonus, certain sections of the code are marked with comments and referred to as bonus areas. These areas contain the code responsible for the multiplayer bonus. This was designed to avoid interfering with the rest of the project's code and to prevent losing points for affecting the main project tasks.)
